# Wordsmith inc

Monorepo for **api** and **web** belonging to Wordsmith

## Run
Run `docker-compose up -d` in terminal

## Database
Postgres v9

## Api

### Swagger
https://localhost:7272/swagger/index.html

### Stack
- .NET 6 
- Entity framwork
- Fluent Validation

## Web

### Stack
- NextJs
- Yup
- framer-motion
- cypress
- tailwind css
